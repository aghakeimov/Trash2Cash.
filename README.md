# Trash2Cash.
Trash2Cash platforması istifadəçilərin çəkdiyi şəkillərlə dolu zibil qutularını GPS ilə xəritədə göstərir. AI məlumatları analiz edərək zibil maşınları üçün optimal marşrut qurur, prioritet təmizləmə planı yaradır və problemli zonaları müəyyən edir. İstifadəçilər düzgün bildirişə görə coin qazanır.
